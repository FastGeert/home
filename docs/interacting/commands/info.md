# Info Commands

Available Commands:

- [info.cpu](#cpu)
- [info.disk](#disk)
- [info.mem](#mem)
- [info.nic](#nic)
- [info.os](#os)


<a id="cpu"></a>
## info.cpu
Takes no arguments.
Returns information about the host CPU types, speed and capabilities


<a id="disk"></a>
## info.disk
Takes no arguments.
Returns information about the host attached disks


<a id="mem"></a>
## info.mem
Takes no arguments.
Returns information about the host memory.


<a id="nic"></a>
## info.nic
Takes no arguments.
Returns information about the host attached NICs, and IPs


<a id="os"></a>
## info.os
Takes no arguments.
Returns information about the host OS.
